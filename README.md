Automated QA Testing System (Secure AQA System)

Description
The Automated QA Testing System is a secure Python-based quality assurance system designed to automate validation, authentication, encryption, and logging processes. The system applies software engineering and security principles to ensure reliable testing, data protection, and system integrity.

This project was developed as part of an academic course and includes full documentation, system diagrams, and security test cases.

Technologies Used
- Python
- Flask (REST API development)
- RESTful APIs
- JSON
- Git & GitHub
- Postman

System Modules
• Authentication Module  
  Manages secure user login, registration, and authentication handling.

• Encryption Module  
  Encrypts sensitive data to ensure confidentiality and protection.

• Validation Module  
  Validates user inputs such as usernames, emails, and text fields.

• Logging Module  
  Logs system activities and errors for monitoring and debugging.

How to Run the Project
1. Clone the repository from GitHub.
2. Navigate to the project directory.
3. Install required dependencies using:
   pip install flask
4. Run the API using:
   python api/app.py

Features
- Secure authentication and authorization
- Input validation to prevent invalid data
- Encryption of sensitive information
- Activity and error logging
- Modular and scalable system architecture
